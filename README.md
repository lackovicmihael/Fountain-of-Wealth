## Running the Project in Android Studio

1. Open the **FountainOfWealth** root directory in Android Studio.
2. Verify that the `app/google-services.json` file is present in the project.
3. Wait for **Gradle Sync** to finish.
4. Run the application on an emulator or a physical Android device.
5. In the Firebase Console, enable **Email/Password Authentication** and configure the **Firestore security rules** described in Section 3.5.
