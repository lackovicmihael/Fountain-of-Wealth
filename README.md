## Pokretanje projekta u Android Studiju 
1. Otvoriti korijensku mapu FountainOfWealth u Android Studiju. 
2. Provjeriti nalazi li se datoteka app/google-services.json u projektu. 
3. Pričekati Gradle Sync. 
4. Pokrenuti aplikaciju na emulatoru ili fizičkom uređaju. 
5. U Firebase Consoleu omogućiti Email/Password prijavu i postaviti Firestore pravila iz poglavlja 3.5.
